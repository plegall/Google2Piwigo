1. Idź do Google Developers Console i połącz się ze swoim kontem Google
  2. Kliknij Create project i nadaj mu nazwę
  3. Kliknij w nowo stworzony projekt
  4. Idź do APIS & AUTH / Credentials
  5. Kliknij Create new Client ID
  6. Podaj nazwę swojej domeny jako Authorized Javascript origins i powyższy Callback URL jako Authorized redirect URI
  7. Teraz skopiuj swój Client ID i Client Secret do konfiguracji Google2Piwigo
  8. Koniec!
