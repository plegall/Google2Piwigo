1. Gå ttill Google Developers Console och logga in med ditt Googlekonto
  2. Klicka påCreate project och ange ett namn
  3. När processen är avslutad, klicka på det projekt du nu skapat
  4. Gå till APIS & AUTH / Credentials
  5. Klicka på Create new Client ID
  6. Ange din domän som Authorized Javascript origins och Callback URL ovan som Authorized redirect URI
  7. Kopiera nu Client ID och Client Secret till Google2Piwigos konfiguration
  8. Klart!
