1. Ísť na Google Developers Console a pripojte sa s Vaším Google účtom
  2. Kliknite Create project a zadajte meno
  3. Akonáhle je proces dokončený, kliknite na novo vytvorený projekt
  4. Ísť na APIS & AUTH / Credentials
  5. Kliknite Vytvorenie nového ID klienta
  6. Uveďte názov domény ako Authorized Javascript origins a vyššie Callback URL ako Autorizované presmerovanie URI
  7. Teraz skopírujte ID klienta a Client Secret do Google2Piwigo konfigurácie
  8. Ste hotový!
