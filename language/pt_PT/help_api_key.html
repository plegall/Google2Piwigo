1. Ir para Google Developers Console e ligue-se com a conta google
  2. Clique Criar projeto e atribua um nome
  3. Quando o processo estiver terminado, clique no projeto criado
  4. Ir para APIS & AUTH / Credencials
  5. Clique Criar um novo ID de cliente
  6. Forneça o nome do seu domínio como Authorized Javascript origins e o acima Callback URL como Authorized redirect URI
  7. Agora copie ID de cliente e o Segredo de cliente para a configuração Google2Piwigo
  8. Já está!
